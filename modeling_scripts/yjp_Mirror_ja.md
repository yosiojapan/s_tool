# yjp_Mirror2.0

## 概要

指定する原点を境にミラーコピーペーストする機能が2種類

1. オブジェクトの反転コピー
2. 同メッシュの反転コピー

## ウィンドウの構成

center : ミラーする座標
UVcenter : UVのセンター
vertexCenter : 選択した頂点をセンターの座標にします
center0 : centerをすべて0にリセット
ラジオボタン : ミラーする軸
merge : ミラーした後の頂点を結合する
UV mmirror : UVもミラーする
FaceMirror : 選択したフェイスをミラーする
SeparateMirror : ミラーしてセパレートする

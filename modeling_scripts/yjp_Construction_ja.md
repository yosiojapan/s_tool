# Construction

## 概要

`Construction`は、選択されたメッシュオブジェクトの中間モデルを編集するツールです。
専用のビューカメラを使用して、バインドされたモデルを動かしながら頂点の移動のみ編集することができます。  

## 使用方法

1. メッシュオブジェクトを選択

2. スクリプトを実行 : S_TOOL > Modeling > Construction

3. ウィンドウの操作

- スクリプトを実行すると、新しいウィンドウが表示されます。
- ウィンドウ内の「Position change」ボタンをクリックすると、ジョイントのY位置が変更されます。
- 「Close」ボタンをクリックすると、ウィンドウが閉じられ、設定がリセットされます。
- ウインドウ内で頂点の移動のみ編集してください。

## 機能詳細

- `選択オブジェクトの処理` :
  - 選択されたメッシュオブジェクトがスキンクラスターを持っているか確認し、持っていない場合はリストから削除します。
  - 各オブジェクトのバックフェイスカリングを設定します。

- `表示レイヤーの設定` :
  - 一時的な表示レイヤーを作成し、選択されたオブジェクトをそのレイヤーに追加します。
  - 必要に応じて新しい表示レイヤーを作成し、オブジェクトを追加します。

- `ビューカメラの設定` :
  - ビューカメラを複製し、ウィンドウ内に表示します。
  - 選択されたオブジェクトを表示し、ビューをフィットさせます。

## 注意点

- `選択オブジェクト` :
  - スクリプトを実行する前に、必ずメッシュオブジェクトを選択してください。
  - スキンクラスターを持つオブジェクトのみが処理されます。

## トラブルシューティング

- `エラーメッセージ` :
  
  - 「Please select a mesh」: メッシュオブジェクトが選択されていない場合に表示されます。
  - 「Select an object with intermediate objects」: スキンクラスターを持つオブジェクトが選択されていない場合に表示されます。

- `表示レイヤーの問題` :

  - オブジェクトが表示レイヤーに追加されない場合は、スクリプトを再実行してください。

## 追加情報

- `関連するプロシージャ` :
  - `yjp_constructionHide` : 中間オブジェクトを非表示にし、レイヤーを削除します。
  - `yjp_constructionPosChange` : ジョイントのY位置を変更します。
